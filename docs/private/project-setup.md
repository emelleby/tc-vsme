This project uses bun as the bundler and runtime. Make sure to use bun instead of node or npm.


## Installation

```bash
bun install
```

## Running the app

```bash
# development
bun run start

# watch mode
bun run start:dev

# production mode
bun run start:prod
```

## Test

```bash
# unit tests
bun run test

# e2e tests
bun run test:e2e

# test coverage
bun run test:cov
